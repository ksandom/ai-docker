# AI-docker

Dockerizing a large collection of AI tools that you can run locally, so that you can get them up quickly.

## Road-map

- [ ] Phase 1: Get one app running on one type of hardware.
- [ ] Phase 2: Get multiple apps running on one type of hardware.
- [ ] Phase 3: Get all supported apps running on multiple types of hardware.
- [ ] Phase 4: Add more apps and hardware support.
- [ ] Phase 5: Connect apps together?
